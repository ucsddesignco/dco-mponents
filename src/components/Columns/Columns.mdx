Column widths are set manually using the `span` prop. The total `span` amount of each column can be of different values but should not exceed the `count`.

## Collapsing Columns

Columns can be made responsive and collapse into a single column when the viewport is below a certain width set by the optional prop `collapseBreakpoint`. The columns below collapse when the viewport is less than or equal to 768px.
